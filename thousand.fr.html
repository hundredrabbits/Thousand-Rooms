Mille Pieces

Mille Pieces

Chat, Hibou, Chauve-Souris & Renard

Illustre par Rekka Bellum
Ecrit par Devine Lu Linvega

Premiere Edition

Vous vous reveillez dans une piece, entoure de murs incassables. Sur un pupitre se trouve un crayon et une feuille de papier. Dans le pupitre, une note portant votre ecriture, vous ne vous souvenez de rien.

Premiere Lecon

Chat se reveille.

Hibou trouve une feuille de papier.

Chauve-souris trouve un crayon.

Renard ouvre le pupitre

Deuxieme Lecon

Chat griffe les murs.

Hibou laisse un message dans le pupitre.

Chauve-souris trouve un autre crayon.

Renard deconstruit le pupitre.

Troisieme Lecon

Chat detruit le crayon et la feuille de papier.

Hibou trouve une note dans le pupitre.

Chauve-souris trouve plusieurs crayons.

Renard reconstruit le pupitre a l'envers.

Quatrieme Lecon

Chat entre dans le pupitre.

Hibou essaie de comprendre.

Chauve-souris n'en peut plus.

Renard a un visiteur.

Cinquieme Lecon

Chat sort du pupitre.

Hibou a tout compris.

Chauve-souris se reveille.

Renard a un visiteur.

Sixieme Lecon

Chat detruit le pupitre.

Hibou trouve un gribouillage.

Chauve-souris trouve un crayon.

Renard a trop de visiteurs.

Derniere Lecon

Que ferais-tu?

Merci
